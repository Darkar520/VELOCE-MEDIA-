Guia de Configuracion de Termux (Habilitacion Unica)

Sigue estos 3 pasos basicos para habilitar el receptor de calidad del portapapeles y el boton Compartir nativo.

1

1. Inicializar Termux

Concede permiso para escribir archivos en el celular e instala Python, FFmpeg y los codecs:

termux-setup-storage && pkg update -y && pkg install python ffmpeg termux-api curl -y && pip install --upgrade yt-dlp spotdl requests beautifulsoup4
2

2. Crear Centinela Inteligente

Descarga el centinela ya listo desde tu repositorio. Lo guarda en la carpeta home de Termux (~) para que siempre lo encuentre:

curl -fsSL -o ~/centinela_veloce.py https://raw.githubusercontent.com/Darkar520/VELOCE-MEDIA-/main/centinela_veloce.py && echo "Descargado OK"
3

3. Encender

Arranca el centinela. Como el curl lo descargo en la carpeta actual, ejecutalo directamente desde ahi:

cd /sdcard/Download && python centinela_veloce.py

Integracion de Boton Compartir Nativo

Puedes anadir Termux al boton "Compartir" de Spotify o Instagram. Ejecuta esto una sola vez en Termux y podras descargar al instante seleccionando Compartir -> Termux en tu telefono: 

mkdir -p ~/bin && cat << 'EOF' > ~/bin/termux-url-opener
#!/bin/bash
url=$1
clear
echo "====================================================="
echo "[*] VELOCE MEDIA ENGINE - INSTANT DOWNLOAD"
echo "====================================================="
if [[ "$url" == *"spotify.com"* ]]; then
    spotdl --output "/sdcard/Download/{list-name}/{artist} - {title}.{output-ext}" --format mp3 "$url"
else
    yt-dlp -f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]" --no-mtime -P "/sdcard/Download" "$url"
fi
echo "[OK] Descarga completada. Presiona una tecla para cerrar."
read -n 1
EOF
chmod +x ~/bin/termux-url-opener
